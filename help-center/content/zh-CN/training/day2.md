# Day 2：实战 + 规模化增长

### 模块 8：创意灵感

#### 目标
用 Shortsmate 的 创意灵感/剧情分析 做选题与改写，建立"选题池 + 热点再创作"流水线，持续稳定产出。

#### 操作建议

- 在 **创意灵感**：从赛道/人群/利益点出发，批量生成选题、标题、钩子（一次 10–20 条）
- 在 **剧情分析**：把"爆款结构"拆成可复用模板（钩子类型、冲突点、信息结构、结尾号召）
- 最终回到 **场景**：把选题落成 30 秒分镜（8–12 镜头）

#### 8.1 赛道筛选

**评分维度（1–5 分）：**

| 维度 | 说明 |
|------|------|
| 需求强度 | 痛点是否明确 |
| 可持续性 | 能否做 30 条不重复 |
| 商业化路径 | 引流/带货/获客 |
| 资源优势 | 素材/人设/产品/渠道 |
| 风险 | 合规/敏感/同质化 |

#### 8.2 热点拆解

**方法：**
- AI 创意，AI 头脑风暴
- 网络搜索

**流程：**

1. **收集：** 同赛道近 7 天高赞/高完播内容 20 条
2. **拆解：** 钩子类型、冲突点、信息结构、镜头节奏、评论区需求
3. **复用：** 只复用"结构与情绪"，不抄具体表达与画面
4. **生成：** 用 AI 输出 10 条"同结构不同主题"的脚本
5. **排期：** 按难度与素材成本排 7 天发布计划

**热点分析提示词（可直接复制）：**

```
我给你{N}条同赛道爆款短视频的标题/简介/脚本要点，请你：
1）按"钩子类型/冲突点/情绪/信息结构/结尾号召"做拆解表；
2）总结可复用的结构模板（至少3个）；
3）基于模板生成10条可拍的新选题（避免侵权与敏感话题）；
4）给每条选题：一句话钩子 + 30秒大纲 + 建议镜头数。
```

### 模块 9：爆款拆解

#### 方法
选取热点，AI 找素材

不复用：不能照搬服用，可识别的标志性创意（避免侵权/抄袭）

#### 课堂动作

1. **选样本：** 同赛道 7 天内高赞/高完播内容 10–20 条
2. **拆结构：** 钩子类型、冲突点、信息递进、结尾号召（填 爆款拆解表）
3. **复用模板：** 总结 3 套"可复用结构模板"（开头句式/中段递进/结尾号召）
4. **再创作：** 每套模板生成 3–5 个新选题（同结构不同主题）

#### 热点解密★★

```
构造采集任务（profiles + 时间）
   ↓
抓 TikTok 视频原始数据
   ↓
逐条视频计算：
   - 发布时间
   - 点赞率
   - 播放速度（VPH）
   ↓
规则引擎判断：
   - Trending
   - Viral
   ↓
打标签 + 结构化输出
   ↓
后续可做：
   - Telegram / Discord 推送
   - 爆款库入库
   - AI 复刻脚本生成
   - 自动剪辑

```

### 模块 10：批量生产

#### 目标
建立可持续的批量生产 SOP

#### 内容工厂 SOP

1. 选题池（每天 10 条备选）
2. 脚本生成（每条 3 个钩子版本）
3. 分镜生成（8–12 镜头，镜头描述与提示词绑定）
4. 关键帧先行（统一风格与角色一致性；角色库全局复用）
5. 视频分段生成（每段多次生成筛选；标注"选中版本原因"）
6. 剪辑模板化（字幕样式/转场/音效/BGM 统一）
7. 封面标题 AB（至少 2 套）
8. 发布与评论运营（置顶评论 + 关键词引导）
9. 复盘迭代（按 STAY/WATCH/ACTION 定位问题）

#### 落地建议

| 要点 | 建议 |
|------|------|
| 角色设定 | 统一角色与风格词，优先解决一致性，再追求镜头花活 |
| 场景 | 按 4–8 秒/段拆分，保证每段都能生成出"可用素材" |
| 视频生成 | 每段至少 2–4 次生成，建立团队统一打分表 |
| 导出 | 素材/成片按"项目-日期-镜头-版本"归档（便于复刻与复盘） |

#### 验收
每组输出：团队《内容生产规范》1 份（命名规则、素材归档、模板复用、审核与合规红线）。

### 模块 11：二次出片

#### 两种做法

**三连发策略：** 同赛道做 3 条不同选题（覆盖不同用户痛点）

**同题材多版本策略：** 同一选题做 2–3 个版本，用于 AB 测试（只改一个变量）

#### 建议的 AB 变量 

- 钩子：结果先行 vs 反差冲突 vs 痛点提问
- 节奏：镜头更碎 vs 信息更密
- 封面：利益点词 vs 反差词
- 号召：评论关键词 vs 关注续集

#### 验收
每组输出：2–3 条成片 v1（可发布）+ 每条标注实验变量 + 发布顺序（哪条先发、间隔多久）。

### 模块 12：热点追踪

#### 目标
把"发出去"变成流程；发布前就明确要测什么、评论区怎么引导。

#### 发布前检查清单（可打印/贴墙）

| 项目 | 要求 |
|------|------|
| 标题 | 结果/利益点明确；包含赛道关键词；避免夸大违规 |
| 封面 | 大字可读（手机 1 秒看懂）；主体突出；不堆字 |
| 话题/标签 | 核心词 2–3 个 + 热点词 1–2 个（谨慎蹭敏感） |
| 评论引导 | 只要一个动作（"评论发'模板'领资料"/"私信关键词"） |
| 发布时间 | 先按你们历史数据；没数据就固定 2 个时段做 A/B |

#### 评论运营脚本（示例）

**置顶评论：** 评论"资料"我发你模板（分镜/提示词/发布清单）

**二次互动：** 你更想看【A】还是【B】？评论区投票，下一条我按票数做。

#### 验收
每组完成：发布清单（标题 3 个版本 + 封面 2 套 + 置顶评论 1 条）。

### 模块 13：数据复盘

#### 目标
用指标驱动迭代，而不是凭感觉；把问题定位到"停留/完播/行动"三段。

#### 复盘指标

| 指标 | 说明 |
|------|------|
| 曝光量/播放量 | 基础数据 |
| 3 秒停留率（STAY） | 钩子效果 |
| 完播率/平均观看时长（WATCH） | 内容质量 |
| 互动率（赞评转藏） | 互动水平 |
| 关注转化/私信转化/点击转化（ACTION） | 转化效果 |
| 主要流量来源 | 推荐关注/搜索/同城 |

#### 迭代原则（简单有效）

| 问题 | 解决方案 |
|------|----------|
| 停留低 | 重做前 3 秒（钩子/字幕/节奏/封面） |
| 完播低 | 中段信息密度与递进不够（删废话、加冲突、加步骤） |
| 行动低 | 结尾号召不清晰（只留一个动作、给理由、给利益） |

#### 验收
每组完成：复盘表模板 + 下一条迭代计划（至少 3 个明确改动）。

## 考核与结业标准

### 必过项

- 交付 30 秒成片（可发布）
- 提交分镜表 + 提示词 + 发布清单 + 复盘表模板

### 评分建议（100 分）

| 项目 | 分数 | 说明 |
|------|------|------|
| 结构与节奏 | 30 | 前 3 秒钩子、信息递进、结尾号召 |
| 画面与一致性 | 25 | 人物/风格稳定，镜头连贯 |
| 声音与字幕 | 20 | 配音节奏、字幕可读、音效加分 |
| 运营可发布性 | 15 | 标题封面、标签话题、评论引导 |
| 可复制性 | 10 | 模板完整、可复用、迭代思路清晰 |

## 合规与风险提示

- **不做侵权搬运：** 不直接下载拼接他人完整视频；不抄袭脚本原句；复用结构不复用表达。
- **不做违规深度伪造：** 未经授权不生成/使用他人肖像或声音进行误导。
- **涉及医疗、金融、教育等领域：** 避免夸大与虚假承诺；必要时走审核流程。
