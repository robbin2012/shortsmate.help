### 模块 1：AI 与短视频趋势

#### 目标
建立 AI 在短视频链路中的"工种地图"，知道哪些环节 AI 最能提效。

#### 内容要点

短视频进入"碎片时间/注意力稀缺"时代；推荐算法推动短视频高速发展。

AI 改变内容创作方式：从"手工创作"到"半自动生产线"。

**AI 在链路中的分工：**

| 类别 | 功能 |
|------|------|
| 内容大模型 | 选题、脚本、分镜、标题、口播稿、提示词生成 |
| 图片大模型 | 关键帧、角色一致性设定、场景设定、风格统一 |
| 视频大模型 | 关键帧动画、运镜、补帧、镜头稳定 |
| 音频模型 | 配音、音色模拟、情绪、节奏控制 |
| 剪辑工具 | 合并剪辑、字幕、音效、封面、模板化导出 |

#### 课堂 Demo
同一个选题，展示"传统做法 vs AI 做法"用时对比（只需要展示：脚本生成、关键帧生成、视频片段生成的速度）。

#### 效率对比

| 环节 | 传统做法（人工） | AI 做法 | 效率差距 |
|------|------------------|---------|----------|
| 脚本生成 | 2～4 小时<br/>（构思、查资料、多轮修改） | 10～20 分钟<br/>（AI生成 + 微调） | 24～48× |
| 关键帧生成 | 1.5～3 小时<br/>（人工分镜、找参考图） | 5～10 分钟<br/>（自动分镜 / 文生图/图生文） | 18～36× |
| 视频片段生成 | 8～16 小时<br/>（布景、道具、拍摄、补拍、剪辑、调色） | 10～20 分钟<br/>（文/图生视频 + 自动剪辑） | 24～48× |
| **总计** | **12～23 小时**<br/>（1–3 个工作日） | **30～50 分钟** | **≈ 30×** |

#### 验收
每组写下：今天要做的成片类型（剧情/口播/混剪/图文转视频/产品故事）。

### 模块 2：模型选型

#### 目标
明确"内容/图片/视频/音频"各用什么模型更稳；理解速度/质量/成本/一致性取舍。

#### 对比维度

| 维度 | 说明 |
|------|------|
| 质量 | 理解力、画面/运动表现、语言 |
| 稳定性 | 成功率、可控性、一致性 |
| 速度 | 迭代频率、生成时长 |
| 成本 | 单次生成成本/套餐 |
| 生态 | 是否方便导出、是否支持工作流/角色库 |

#### 模型/类别示例

| 类别 | 示例 |
|------|------|
| 内容大模型 | GPT（OpenAI）/ Gemini（Google）/ Qwen（Alibaba） |
| 图片大模型 | GPT Image（OpenAI）/ Nano Banana（Google）等 |
| 视频大模型 | Veo（Fast/Quality）/ Sora / Kling / Hailuo / MiniMax / Seedance 等 |

#### 推荐选型策略

1. **快出片（迭代优先）**：内容模型（稳定输出脚本）+ 视频模型（Fast 档）+ 剪辑模板
2. **精品画面（质感优先）**：图片模型（先把关键帧做稳）+ 视频模型（Quality 档）+ 少镜头高质量
3. **剧情连贯（叙事优先）**：内容模型（分镜详细）+ 角色设定卡（强约束）+ 多次生成筛选

> **说明：** 本课主工具为 shortsmate.app，其内部可对接多类模型（PPT 中列举的内容/图片/视频模型）。本模块只讲"怎么选、怎么搭配、怎么避坑"，不把时间花在逐个工具的细枝末节。

#### 课堂练习：选型卡
每组填：内容模型、图片模型、视频模型、配音方式、剪辑工具、备选方案。

#### 验收
每组提交"工具组合 + 风险点"（例如：人物不一致、运镜乱、时长不够、中文口型）。

### 模块 3：AI 工具上手

#### 目标
用半天把"内容/图片/视频/音频"各类 AI 工具快速上手到够用水平；掌握能迁移到 shortsmate 的提示词与一致性方法。

#### 3.1 提示词总公式

```
你是谁/你要做什么（角色/任务）
目标（受众/平台/时长/风格）
结构（前3秒钩子-中段推进-结尾号召）
内容约束（必须出现/禁止出现/敏感规避）
镜头语言（景别/运镜/节奏/转场）
输出格式（表格/分镜/逐句口播/时间码）
```

![分镜创建](/images/training/image_1.png)

点击蓝色箭头即可创建分镜，点击黄色箭头即可描述该分镜内容。

#### 3.2 30 秒脚本提示词

```
请为【赛道：{赛道}】【受众：{受众}】生成一条【30秒竖屏短视频】脚本。
要求：
1）结构：1-3s吸睛开篇；4-24s高能叙事/信息密度；25-30s强力号召（关注/评论/私信/下单）。
2）用"用户视角"，娱乐化包装但不低俗；语言口语化、节奏快。
3）输出为表格：时间段、画面（镜头/场景/人物动作）、台词/旁白、字幕要点、音效/音乐建议。
4）给3个不同钩子版本（开篇3秒）。
```

![提示词优化](/images/training/image_2.png)

点击提示词优化，即可对画面对人物进行创意

#### 3.3 角色设定卡

模板（每组填一次，全局复用）：

| 项目 | 内容 |
|------|------|
| 角色名 | |
| 年龄/性别/体型/气质 | |
| 发型/五官特征/正面/侧面/后面 | |
| 服饰（固定主色/配饰） | |
| 禁止变化项（必须一致） | |
| 可变化项（允许变化） | |
| 参考图（可选） | |

![角色三视图](/images/training/image_3.png)

切换到美术设计，输入识别到的描述或者ai优化过的文字，点击图片生成即可得到人物的三视图，表情图，细节图，也可以自行上传，如电商上传自己的产品。

#### 3.4 批量生成与筛选

**统一打分表（每个镜头 1–5 分）：**

- 清晰度/噪点
- 人物一致性
- 情绪与动作符合脚本
- 运镜稳定/不穿帮
- 与上一镜头连贯

#### 验收
每组提交：角色设定卡 + 30 秒脚本（表格版）+ 关键帧清单（8–12 镜头）。

### 模块 4：项目搭建

#### 目标
能看懂 shortsmate 的核心页面与流程，并在 50 分钟内把"项目骨架"搭好（角色/风格/场景/分镜入口都具备）。

![场景创建](/images/training/image_4.png)

手动点击+号创建场景，把整个项目分镜都分镜出来

#### 工具的页面模块介绍

**顶部流程页签（常见）：** 场景、美术设计、视频生成、剧情分析、创意灵感

**核心资产：** 角色设定（统一角色管理、全局可调用，消除角色变化不确定性）

**核心生产：** 关键帧创作（从脚本说明生成关键帧图）→ 关键帧动画（生成视频，多次生成择优）→ 合并剪辑出片

**导出：** 右上角 导出（发布前的素材/成片输出）

#### 讲师演示

1. **新建项目：** 命名规则（赛道-主题-日期-版本）
2. **建立角色库：** 导入/生成参考，写"资产描述"（角色设定卡落到系统里）
3. **设置风格与世界观：** 统一风格词（如 3D 写实、皮克斯感、电影光线等）
4. **建立场景列表：** 按 8–12 镜头拆场景（每个场景 4–8 秒），先跑通再精修

#### 学员实操
每组完成：1 个项目 + 1 组主角角色 + 1 个配角（可选）+ 3 个基础场景模板

#### 验收
每组现场展示：角色设定卡已经"系统可调用"，并能在场景里一键引用。

### 模块 5：分镜与关键帧

#### 目标
把"结构"和"流程"固化成 SOP，让团队可以规模化复制。

#### 5.1 30 秒结构

**STAY（停留）：** 前 1–3 秒必须解决"为什么你要看下去"

**WATCH（观看）：** 中段用冲突/反差/递进信息密度提升完播

**ACTION（行动）：** 结尾明确号召（评论关键词/关注/私信/领资料/进群）

**建议节奏：**

| 时间段 | 内容 |
|--------|------|
| 1–3s | 钩子（反差/结果先行/痛点击穿/挑战） |
| 4–12s | 给理由/给证据（1–2 个点） |
| 13–24s | 升级冲突/方法论/步骤（2–3 个点） |
| 25–30s | 总结 + 行动号召（必须单一明确） |

#### 5.2 制作流程 SOP

1. 选题与定位（赛道/人群/利益点）
2. 脚本与分镜（按时间码输出）
3. 关键帧（先做"可控的图"，再做"不可控的视频"）
4. 关键帧动画（多次生成择优，必要时拆分镜头）
5. 合并剪辑（节奏、转场、BGM、音效）
6. 配音与字幕（口播节奏、关键词加粗、字幕可读性）
7. 封面与标题（测试 2–3 套）
8. 发布与复盘（指标、评论引导、二创/续集规划）

#### 验收
每组输出：8–12 镜头分镜表（含镜头描述与生成提示词）。

#### 5.3 分镜与关键帧实操

**组内分工（建议）：**

| 角色 | 职责 |
|------|------|
| A 角色 | 脚本/分镜负责人（盯叙事与节奏） |
| B 角色 | 画面负责人（关键帧/一致性/风格） |
| C 角色 | 剪辑与声音负责人（配音/字幕/节奏） |

**实操步骤（Shortsmate）：**

1. 在 场景 页把 30 秒拆为 8–12 镜头（或 6–8 镜头也可，先稳再加）
2. 在 美术设计 / 关键帧创作：为每个镜头补齐"脚本说明"（人物/地点/时间/动作/情绪/镜头语言）
3. 勾选/引用全局 角色设定 与场景设定，统一风格词
4. 生成关键帧图：同镜头多次生成，用打分表筛选最佳

**关键帧要求：**

- 8–12 张关键帧，尽量统一风格/色彩/角色
- 每张对应分镜表 1 行（镜头描述清楚：景别、角度、动作、情绪、光线）

**关键帧提示词**

```
竖屏9:16，电影感光线，真实质感，{角色设定卡关键特征}，
场景：{地点/时间/天气}，
镜头：{景别/角度/构图}，
动作：{角色动作}，情绪：{情绪}，
色彩：{主色/对比}，
不要：变脸/畸形手/多余人物/水印/文字
```

#### 验收
每组完成：关键帧 8–12 张（通过打分表筛选）+ 每张对应的镜头说明（可复刻）。

### 模块 6：视频生成

#### 目标
用 shortsmate 的 关键帧动画/视频生成把关键帧变成可用的视频片段，并建立"多次生成择优"的课堂标准。

#### 操作要点

在 视频生成 里按"场景 1/场景 2/场景 3..."分段生成

![视频生成](/images/training/image_5.png)

关注生成参数：模型档位（如 Fast/Quality）、分辨率、时长（建议 4–8 秒/段）

![生成参数](/images/training/image_6.png)

每段至少生成 2–4 次，按打分表选择最优（不要指望一次完美）

#### 策略

- 先把 8–12 张图分成 3–6 段视频（每段 4–8 秒），再剪到 30 秒
- 每段至少生成 2–4 次，按打分表选最优

#### 动画提示词（示例，可用于每段视频描述）

```
基于该关键帧生成4秒视频：轻微推镜，人物微表情变化，背景轻微景深变化；
保持人物外观一致；动作自然不夸张；避免画面抖动、变形、穿帮。
```

#### 验收
每组输出：3–6 段视频片段（总时长 >= 30 秒素材量），并标注"已选最佳版本"的原因。

### 模块 7：剪辑与字幕

#### 目标
把 shortsmate 生成的片段合成 30 秒成片 v1，并具备发布所需的"封面/标题/评论引导"。

#### 剪辑清单

| 要素 | 要求 |
|------|------|
| 镜头节奏 | 前 3 秒必须快；中段信息密度高但不乱 |
| 字幕 | 关键字突出（颜色/加粗/放大）；每屏不超过 2 行 |
| 音效 | 开篇"提示音/转场音"增强停留；结尾"号召音"增强行动 |
| 封面 | 一句话利益点 + 反差词；人物/主体清晰 |

#### 验收
成片 v1：竖屏 9:16、25–35 秒、含字幕与配音、可发布画面（无明显穿帮/违规元素）。
