# 支持的模型

ShortsMate 内部对接多类模型，主要覆盖脚本、画面与视频三个环节。

| 模型类型 | 主要用途 | 示例参考 |
| --- | --- | --- |
| 脚本/内容模型 | 生成话题、脚本、钩子、分镜初稿 | OpenAI GPT、Gemini、Qwen |
| 图像/关键帧模型 | 生成关键场景、角色、风格一致的画面 | OpenAI GPT Image、Nano Banana |
| 视频模型 | 生成片段、动画、镜头稳定与补帧 | Veo（Fast/Quality 档）、Sora、Kling、Hailuo、MiniMax、Seedance、Jimeng、Grok |

各个模型可以一键切换，不同型号对应不同价格，通常低版本/低分辨率的模型成本更低。建议先用这些更经济的模型快速验证方向，确认满足需求后再升级到更高档的模型以提高质量。

## 如何选择模型

- 首先明确目标（如速推、质感、叙事）后选定内容/画面/视频模型组合
- 在模型档位（Fast/Quality）和分辨率之间权衡，满足产出 && 成本
- 不确定时先用默认模型验证方向，再逐步升级高阶模型

## 模型更新

模型与能力会持续更新，最新列表以产品内展示为准。
